# Product Overview

Claude Code学習用リポジトリ。AI支援開発ツール（Claude Code）の使い方とKiro風Spec駆動開発（cc-sdd）の実践を学ぶためのサンプルプロジェクト。

## Core Capabilities

- **Claude Code学習**: 3つのモード（Ask before edits、Edit automatically、Plan mode）の理解と実践
- **Spec駆動開発**: cc-sddを使用したKiro風の仕様駆動開発ワークフローの体験
- **MCPサーバー連携**: カスタムMCP設定によるClaude Codeの拡張

## Target Use Cases

- Claude Codeを初めて使用する開発者の学習
- Spec駆動開発（SDD）ワークフローの理解と実践
- AIエージェントとの効果的な協働パターンの習得

## Value Proposition

実際に動作するサンプルアプリ（htmx Todo App）を通じて、Claude CodeとSpec駆動開発の両方を体験できる学習環境を提供。
