# 業界一覧（四季報 業界地図ベース）

東洋経済「会社四季報 業界地図 2026年版」（194業界収録）を参考にした業界分類。
12大分類と主要な中分類・小分類を体系的に整理。

---

## 使い方

1. 大分類から興味のある領域を選ぶ
2. 中分類で具体的な業界を絞り込む
3. 小分類でさらに深掘りしたいセグメントを特定する
4. 選んだ業界に対してリサーチを実行する

## 横断的な注目テーマ（全業界共通）

| テーマ | 関連する課題 |
|--------|------------|
| **生成AI** | 業務効率化、自動化、人材への影響 |
| **カーボンニュートラル** | 脱炭素、再エネ、排出削減 |
| **人手不足** | 高齢化、採用難、働き方改革 |
| **地政学リスク** | サプライチェーン、安全保障、経済安保 |
| **DX** | レガシー刷新、データ活用、デジタル人材 |
| **サイバーセキュリティ** | 情報漏洩、ランサムウェア、ゼロトラスト |
| **ESG/SDGs** | サステナビリティ、人権、ガバナンス |
| **規制変化** | 個人情報、AI規制、業界固有法規制 |
