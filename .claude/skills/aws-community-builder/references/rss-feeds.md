# AWS RSSフィード一覧

AWS最新情報をキャッチアップするためのRSSフィード一覧。
AI Engineering カテゴリに特に関連度が高いものは **[優先]** マークを付与。

## 使い方

1. スキル起動時に、優先フィードの最新記事を WebSearch/WebFetch で確認する
2. AI/ML 関連の新サービス・アップデートを抽出する
3. ブログネタ候補としてリストアップする

## フィード一覧

### 最優先（毎回チェック）

| フィード名 | URL | 翻訳要否 | 備考 |
|-----------|-----|---------|------|
| **[優先] AWS Machine Learning Blog** | https://aws.amazon.com/jp/blogs/machine-learning/feed/ | 英→日 | AI/ML直結。最重要フィード |
| **[優先] AWS Blog（メイン）** | https://aws.amazon.com/blogs/aws/feed/ | 英→日 | 新サービス発表の第一報 |
| **[優先] What's New with AWS** | https://aws.amazon.com/jp/about-aws/whats-new/recent/feed/ | 不要 | 全般的な新着情報 |
| **[優先] AWS Japan Blog** | https://aws.amazon.com/jp/blogs/news/feed/ | 不要 | 日本語。国内向け情報 |

## キャッチアップ時の検索クエリ例

RSSフィードに加えて、以下のWebSearchクエリも活用する：

```
AWS Bedrock new features {year} {month}
AWS SageMaker updates {year}
AWS generative AI announcements {year}
Amazon Q new capabilities {year}
AWS AI ML new services {year}
AWS re:Invent {year} announcements AI
site:aws.amazon.com/blogs/machine-learning/ {year}
```
