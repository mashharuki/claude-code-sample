## 注目トレンド

- **AIエージェントの全面的な普及**: Amazon Connect Health（ヘルスケア向け5つのAIエージェント）、Bedrock AgentCore Policy（エージェントのガバナンス）、OpenClaw（プライベートAIエージェント）、Strands Labs（エージェント開発実験）と、AIエージェント関連の発表が圧倒的に多い。AWSは「エージェント=次世代の計算単位」と位置づけ、推論基盤を新たなビルディングブロックとして強調している。

- **OpenAIパートナーシップによるAIエコシステム再編**: 500億ドル投資、Stateful Runtime Environment、OpenAI Frontierの独占クラウド配信など、OpenAIとの大型提携がAWSのAI戦略を大きく転換。Bedrock上でOpenAIモデルが利用可能になることで、マルチモデル戦略がさらに加速。

- **セキュリティのマルチクラウド統合**: Security Hubがマルチクラウド対応に拡張され、OCSF準拠のクロスドメイン相関分析が可能に。大企業のハイブリッド・マルチクラウド環境を前提としたセキュリティアーキテクチャへの転換が明確。

- **メディア/動画のAI自動変換**: Elemental Inferenceの登場により、ライブ動画のSNS向け自動変換（縦動画化、ハイライト抽出）がマネージドサービスとして提供開始。コンテンツ制作ワークフローのAI化が加速。

- **コスト最適化の拡充**: Database Savings PlansのOpenSearch/Neptune対応、VPC暗号化の有料化など、コスト管理に影響する変更が複数あり、FinOps観点での注目が必要。
