## 注目トレンド

- **AIエージェントの本格的なエンタープライズ展開**: Amazon Connect Health（ヘルスケア5エージェント）、Bedrock AgentCore Policy（ガバナンス）、Strands Labs（実験基盤）と、AIエージェントのライフサイクル全体をカバーする発表が集中。エージェントAIが実験段階から本番運用フェーズに移行していることを示している。

- **OpenAIとの$500億パートナーシップによるAIプラットフォーム戦略の転換**: AWSがOpenAI Frontierモデルの独占クラウド配信プロバイダーとなり、Trainium3/4チップでの大規模計算基盤を提供。自社モデル（Claude on Bedrock等）だけでなく、他社の最先端モデルもホストする「AIスーパーマーケット」戦略が鮮明に。

- **セキュリティのマルチクラウド化**: Security Hubのマルチクラウド拡張は、AWSが「AWS環境だけを守るツール」から「企業全体のセキュリティプラットフォーム」への進化を目指していることを示す重要なシグナル。CrowdStrike、Okta、Splunkなどとのパートナー統合も注目。

- **AI駆動の運用自動化**: Elastic BeanstalkのAI環境分析、ECS/EKS/ServerlessのMCPサーバーなど、既存AWSサービスへのAI機能組み込みが加速。開発者体験の向上とオペレーション自動化が同時に進行。

- **日本コミュニティの活発化**: JAWS DAYS 2026が1,500名超の参加で世界最大のAWS Community Dayとなり、フィジカルAI開発支援プログラムなど日本独自の施策も展開。AWSジャパンのコミュニティ投資が拡大中。
