# AWS CDK カンファレンスのcfpの要件

このイベントは、 AWS CDK に関するあらゆる知見を共有するためのイベントです。

トークのテーマはAWS CDK、CDK Terrain(CDKTF)、cdk8s(+)に関連する話であれば何でもアリです！
今回のイベントも昨年と同様に初心者/上級者ともに楽しめるイベントをコンセプトとしているので、どなたでも気軽にご応募ください。

JAWS-UG CDK支部は、あなたならではの経験や、愛と情熱の込められたトークを歓迎します。

【公募期間】
2026/04/15（水）～ 2026/05/31（水）23:59 JST

【募集要項】
セッション時間を LT、10分、20分 から選んで応募してください。
1人で複数応募も可能です。
複数応募された場合、原則として重複して採択されることはありません。
採択の可否は 6/5 夜（予定）にメールにてご連絡します。
6/12(金) までに登壇可否をご返信ください。
返信していただけない場合は不参加とみなす場合があります。
旅費の補助や、講演費などの支給はありません。あらかじめご了承ください。

【その他留意事項】
会場は登壇用のメイントラック、サブトラック、ワークショップトラックの3つを予定しています。
この CfP はメイントラック、サブトラックのセッション募集になります。メイントラック、サブトラックの振り分けは運営側で行います。
オンライン配信はメイントラックのみになります。

ワークショップトラックに参加される方でもメイントラック、サブトラックに応募していただけます。
ワークショップへ参加を希望する方は是非会場までお越しください。
会場にも設備はありますが、念の為、ご自身のPCからHDMIに接続するためのアダプタは各自お持ちください。
